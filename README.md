# my-tiddlywiki-xp

## 项目简介

`my-tiddlywiki-xp` 是一个基于 [TiddlyWiki XP](https://keatonlao.github.io/tiddlywiki-xp/) 构建的个人知识管理系统。原始来源为 [TiddlyWiki XP GitHub](https://github.com/keatonlao/tiddlywiki-xp)。

该项目已经通过 TiddlyWiki 的升级工具进行了更新，并对相关插件进行了必要的升级。最终版本重新保存为单文件 HTML 格式的 TiddlyWiki，便于本地使用和分享。

## 更新说明

- **升级**：已经通过 TiddlyWiki 的 `upgrade` 工具对项目进行了升级，确保与最新的 TiddlyWiki 版本兼容。
- **插件更新**：已更新项目中使用的所有插件，以提供更好的功能和兼容性。
- **格式**：将文件重新保存为单文件 `.html` 格式的 TiddlyWiki，简化了部署和使用流程。

## 如何使用

1. 下载或克隆此项目。
2. 打开 `my-tiddlywiki-xp.html` 文件，开始使用你的个人知识管理系统。
3. 编辑和添加内容，TiddlyWiki 会自动保存所有更改。
